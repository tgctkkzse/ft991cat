# Yaesu FT-991 コントローラ
FT-991Mが手に入ったので、パソコンから操作できるようにしてみました。  
wfviewと同じように、com0comで他のソフトと連携できるようにしてあります。  
IC-705 + wfview + WSJT-Xと同じように  
FT-991 + ft991control + WSJT-Xで動作させています。
